Security Codes Help

Where do I find the codes?

Keystone Adventures sends the codes to the school at the same time as we send the link.

If you didn't receive the codes yet, ask the trip organizer.

What do the codes look like?

The trip code is all digits, usually 3 digits in length, something like 143.

The security code can have digits and letters, and it is always 6 characters long.

If you think you have the right codes, but something is not working, try refreshing the page, then make sure you don't add any extra spaces when entering the codes. Try sending them again

If after all you still have problems, please contact your trip organizer.
